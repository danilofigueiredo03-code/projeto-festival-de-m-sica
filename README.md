O Projeto Festival de Música foi desenvolvido e criado individualmente por mim com o objetivo de simular a divulgação de um festival musical fictício por meio de uma plataforma digital. O site foi criado para apresentar o evento, suas principais atrações e informações relevantes para o público, utilizando conceitos de design e desenvolvimento web voltados para a promoção de eventos. O festival fictício conta com a participação de grandes artistas e bandas, como NX Zero, Bruno Mars, Billie Eilish e Linkin Park, destacando diferentes estilos musicais para atrair diversos públicos. O projeto teve como finalidade desenvolver habilidades em criação de sites, organização de conteúdo, design de interfaces e comunicação visual, simulando um cenário real de divulgação e marketing de eventos musicais.
